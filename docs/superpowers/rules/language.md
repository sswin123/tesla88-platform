# 语言规则 — 项目永久规则

## 规则

所有用户可见的内容必须使用**简体中文**。

## 覆盖范围

- 思考过程说明
- 计划和步骤说明
- 工具调用前的说明文字
- 进度更新
- 任务完成报告
- 错误说明
- 向用户提问
- 审查结果
- Claude 内部进度文字（在对话中显示的所有文字）

## 严禁

| 语言 | 示例 |
|------|------|
| 日文 | 次に、作成します、完了しました、変更ファイル、修正済み、テスト結果 |
| 韩文 | 대기、완료、보고 |

## 必须使用中文

| 禁止 | 使用 |
|------|------|
| 次に | 下一步 ✅ |
| 作成します | 创建 ✅ |
| 完了しました | 完成 ✅ |
| 変更ファイル | 修改文件 ✅ |
| 修正済み | 已修复 ✅ |
| テスト結果 | 测试结果 ✅ |
| 완료 | 完成 ✅ |
| 대기 | 等待确认 ✅ |

## 例外（允许保持英文）

仅限以下技术术语：

- 代码内容
- 文件名
- API 名称
- function name
- commit hash
- 数据库字段名
- 命令行命令

## 重要说明

Claude 内部进度文字（在对话 UI 中显示的文字）同样适用此规则。
上下文重载、压缩后必须立即重新检查此规则。
